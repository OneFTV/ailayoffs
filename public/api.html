Endpoints

GET /api/layoffs

Returns layoff events. Supports filtering.

ParamTypeDescription
sectorstringFilter by sector (e.g. "tech", "finance")
companystringFilter by company name
fromdateStart date (YYYY-MM-DD)
todateEnd date (YYYY-MM-DD)
{
  "data": [
    {
      "company": "Acme Corp",
      "sector": "tech",
      "jobs_displaced": 1200,
      "date": "2026-01-15",
      "confidence": "confirmed",
      "source": "SEC Filing"
    }
  ],
  "total": 342,
  "page": 1
}

GET /api/stats

Aggregate statistics.

{
  "total_displaced": 487000,
  "companies_affected": 312,
  "countries": 48,
  "avg_per_month": 24350
}

GET /api/companies

List of all tracked companies with displacement totals.

GET /api/by-country

Displacement data grouped by country.

GET /api/timeline

Monthly displacement counts for charting.

{
  "timeline": [
    {"month": "2025-01", "displaced": 18200},
    {"month": "2025-02", "displaced": 22400}
  ]
}

Authentication

Include your API key in the header:

Authorization: Bearer YOUR_API_KEY
